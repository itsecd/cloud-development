<details>
<summary> «Транспортное средство» </summary>
Необходимо сгенерировать информацию о транспортном средстве со следующими характеристиками:


| №  | Название                          | Тип данных |
|---:|-----------------------------------|-----------|
| 1  | Идентификатор в системе           | int       |
| 2  | VIN-номер                         | string    |
| 3  | Производитель                     | string    |
| 4  | Модель                            | string    |
| 5  | Год выпуска                       | int       |
| 6  | Тип корпуса                       | string    |
| 7  | Тип топлива                       | string    |
| 8  | Цвет корпуса                      | string    |
| 9  | Пробег                            | double    |
| 10 | Дата последнего техобслуживания   | DateOnly  |

Комментарии к характеристикам:
- VIN-номер берется из раздела Vehicle
- Производитель берется из раздела Vehicle
- Модель берется из раздела Vehicle
- Тип корпуса берется из раздела Vehicle
- Тип топлива берется из раздела Vehicle
- Цвет корпуса берется из раздела Commerse
- Год выпуска не может быть позже текущего
- Пробег не может быть меньше нуля
- Дата последнего техобслуживания не может быть раньше года выпуска
</details>
